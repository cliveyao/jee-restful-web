# JEE RESTful Web

RESTful Web 服务的简单实现，目前实现了注册、认证、用户管理等简单功能。

## 技术

1. 后端

    使用Maven构建，主要使用的框架

    * Spring Framework 4.1
    * Apache Shiro 1.2
    * Spring MVC 4.1
    * MyBatis 3.3
    * Alibaba Druid 1.0
    * Ehcache 2.6
    * SLF4J 1.7、Log4j 2.3
    * Spring HATEOAS 0.17


2. [前端](https://github.com/howiefh/restful-web-app)

    使用Gulp构建，Bower进行包管理，主要使用的框架

    * Bootstrap
    * AngularJS
    * Restangular
    * AngularUI Router
    * UI Bootstrap
    * Angular-sanitize
    * AngularJS ui-select
    * xtform
